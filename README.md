## Project Setup

### Prerequisites

* Install Docker for desktop [Visit this link](https://www.docker.com/products/docker-desktop)
* Install Node JS/NPM [Visit this link](https://nodejs.org/en/)
* NoSQL Workbench for DynamoDB [Visit this link](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/workbench.html)


## Authentication Service

### Steps:

* Go to the project folder and install node dependencies
    ```bash
    cd authentication-service
    npm i
    ```
* Create a Docker Container (Make sure Docker Desktop is running)
    ```bash
    npm run dc-up
    ```
* Setup Dynamo DB
    ```bash
    npm run setup:dynamodb
    ```
* Run the server
    ```bash
    npm start
    ```
## Order Service / Tenant Service / Cart Service

### Steps:

* Create a Docker Container (Make sure Docker Desktop is running)
    ```bash
    # Run this command in the root
    npm run up
    ```

* Go to the project folder and install node dependencies
    ```bash
    cd order-service
    npm i
    ```
* Setup Dynamo DB
    ```bash
    npm run setup:dynamodb
    ```
* Run the server
    ```bash
    npm start
    ```

## Product Service

### Steps:
